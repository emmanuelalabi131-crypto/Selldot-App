# Selldot App

Selldot is a digital platform designed to provide secure and transparent property transactions in Nigeria.

## Problem
Many Nigerians lose land and property due to fraud and lack of verification systems.

## Solution
Selldot introduces a secure digital system that helps verify land ownership and prevent fraud.

## Features
- Property verification system
- Secure transaction tracking
- User-friendly interface

## Motivation
This project was inspired by a real-life experience where my family lost land due to fraud.
